## 🚀 快速入门

### 系统要求
- **操作系统**: Windows 10/11 (推荐) / Linux / macOS
- **Python版本**: 3.8或更高版本
- **内存**: 8GB以上 (推荐16GB)
- **存储**: 至少5GB可用空间
- **网络**: 稳定的互联网连接

### 安装步骤

1. **下载项目**
   ```
   git clone https://github.com/your-username/AI_NovelGenerator.git
   cd AI_NovelGenerator
   ```

2. **安装依赖**
   ```
   pip install -r requirements.txt
   ```

3. **配置API密钥**
   - 复制 `config.example.json` 为 `config.json`
   - 编辑配置文件，添加您的LLM服务API密钥

4. **启动应用**
   ```
   python main.py
   ```

### 第一次使用

1. **配置LLM服务**
   - 打开"配置"标签页
   - 选择您要使用的LLM服务
   - 输入API密钥和相应配置

2. **创建第一个项目**
   - 在"小说参数"标签页设置小说基本信息
   - 点击"新建小说项目"

3. **开始创作**
   - 按照界面引导逐步创建世界观、角色、章节等

### 3. 基本参数设置

#### Temperature（温度）
- **0.1-0.3**: 输出更加确定和一致，适合技术文档
- **0.4-0.7**: 平衡创造性和一致性，推荐小说创作使用
- **0.8-1.0**: 更具创造性，可能产生不一致内容

#### 最大令牌数
- **短章节**: 2000-4000
- **标准章节**: 4000-8000
- **长章节**: 8000-16000

| 服务 | 时间窗口 | 请求数限制 | 建议间隔 |

| DeepSeek | 1分钟 | 100次 | 0.6秒 |
| 智谱AI | 5小时 | 600次 | 33秒 |
| Gemini | 1分钟 | 60次 | 1秒 |

### 5. 章节内容生成

#### 生成流程

1. **内容检索**
   - 自动检索相关历史章节
   - 获取角色当前状态
   - 分析世界观相关设定

2. **提示构建**
   - 整合章节蓝图信息
   - 融入上下文信息
   - 构建详细的生成提示

3. **内容生成**
   - 调用LLM API生成内容
   - 自动验证内容完整性
   - 智能重试和修复

4. **质量检查**
   - 零容忍省略检查
   - 一致性验证
   - 字数统计和调整

#### 内容质量控制

**自动验证规则**:
```
禁止内容:
- 包含省略号（...、…等）
- 内容中断或未完成
- 明显的复制粘贴
- 不符合设定的内容

必须内容:
- 完整的章节结构
- 符合字数要求
- 保持风格一致
- 遵循时间线逻辑
```

**质量指标**:
```
完整性: 100% (无省略内容)
一致性: 95%+ (符合设定和时间线)
原创性: 90%+ (避免重复内容)
可读性: 85%+ (流畅自然)
逻辑性: 90%+ (情节合理)
```

## 🔧 故障排除

### 常见错误及解决方案

#### 1. API连接失败
```
错误信息: "API调用失败: Connection timeout"
可能原因:
- 网络连接问题
- API密钥错误
- 服务暂时不可用

解决方案:
1. 检查网络连接
2. 验证API密钥格式
3. 确认服务状态
4. 增加超时时间
5. 切换备用API服务
```

#### 2. 生成内容不完整
```
错误信息: "章节生成被截断"
可能原因:
- 超过最大令牌数限制
- API响应中断
- 内容包含省略号

解决方案:
1. 增加最大令牌数设置
2. 启用自动重试机制
3. 检查网络稳定性
4. 分段生成长内容
5. 启用内容修复功能
```

## ❓ 常见问题

### 1. 基础使用问题

**Q: 如何更改已设置的API密钥？**
A: 进入"配置"标签页，选择对应的LLM服务，修改API密钥后点击"保存配置"。

**Q: 可以同时使用多个LLM服务吗？**
A: 可以。在配置中添加多个LLM服务，生成时可以在"章节"标签页选择使用哪个服务。

**Q: 生成的章节可以手动编辑吗？**
A: 可以。在"章节"标签页选择已生成的章节，点击"编辑"按钮进行修改。

**Q: 如何调整生成内容的风格？**
A: 在"小说参数"标签页设置创作风格，或通过Temperature参数调整创造性程度。

### 2. 技术问题

**Q: 为什么生成的章节只有标题？**
A: 可能是触发了零容忍验证（如包含省略号），系统会自动重试或提示错误。

**Q: API调用频率超限怎么办？**
A: 系统会自动处理频率限制，也可以在配置中调整调用间隔时间。

**Q: 向量数据库占用空间很大怎么办？**
A: 可以在"设置"标签页清理向量数据库，或调整数据保留策略。

**Q: 如何提高生成速度？**
A: 增加并发数、使用更快的模型、减少生成内容长度都可以提高速度。

### 3. 创作相关

**Q: 如何保持小说风格的一致性？**
A: 在"小说参数"中明确设置风格要求，系统会自动维持风格一致性。

**Q: 角色行为不符合设定怎么办？**
A: 检查角色档案是否完整，可以使用一致性检查功能修正。

**Q: 如何处理情节逻辑漏洞？**
A: 使用自动一致性检查，发现问题后手动修正或重新生成相关章节。

**Q: 可以导入已有的小说内容吗？**
A: 目前支持文本格式导入，可以在"章节"标签页使用"导入章节"功能。

### 4. 高级功能

**Q: 如何自定义一致性检查规则？**
A: 在"设置"标签页可以添加自定义检查规则，需要一定的编程知识。

**Q: 知识库如何更新？**
A: 知识库会定期自动更新，也可以手动在"设置"中强制更新。

**Q: 模板推荐不准确怎么办？**
A: 可以在推荐后手动调整，系统会学习用户偏好，提高推荐准确性。

**Q: 如何优化批量生成的效果？**
A: 根据项目特点选择合适的批量大小，启用自动质量检查和修复。
