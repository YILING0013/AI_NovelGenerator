# 🔧 节标题重复和完整性问题修复

## 用户反馈问题

**现象**：
1. 第1章中"基础元信息"出现了4次
2. 从第2章开始内容不足，偷工减料，省略了后面的节
3. 格式混乱

**用户原话**："第一章里为什么基础元信息有4章？从第二章开始就没有正常了？但是从第2章开始项目就不足了开始偷工减料。"

## 根本原因分析

### 问题1：Prompt未明确禁止节标题重复

**之前的Prompt**：
```
每个章节必须遵循以下结构（按顺序）：
   ### **第X章 - [章节标题]**
   ## 1. 基础元信息
   ...
   🚨 **格式禁忌**：
   - **严禁**在"基础元信息"中重复写"第X章 - 标题"
```

**问题**：
- 只禁止了"第X章 - 标题"的重复
- 没有禁止"## 1. 基础元信息"这个**节标题**本身的重复
- LLM可能误解为可以多次使用同一个节标题

### 问题2：Prompt未强调每章必须包含所有节

**之前的Prompt**：
```
2. **完整性铁律**：
   - 禁止任何省略（如"..."或"略"）。
   - 每章至少800字详细描述。
   - 必须包含【匠心思维应用】。
```

**问题**：
- 只明确要求了"匠心思维应用"
- 没有明确列出所有7个必需的节
- LLM可能认为后面的节可以省略

### 问题3：缺少节级别的验证

**之前的验证逻辑**：
- 只检查章节标题是否重复
- 只检查章节内容长度
- 没有检查节标题是否重复
- 没有检查是否缺失节

## 修复方案

### 修复1：明确禁止节标题重复

**位置**：`novel_generator/blueprint.py:419-444`

**修复后的Prompt**：
```
1. **章节格式规范**：

每个章节必须遵循以下结构（按顺序，**每个节标题只能出现一次**）：

   ### **第X章 - [章节标题]**
   ## 1. 基础元信息
   ...
   ## 2. 张力与冲突
   ## 3. 匠心思维应用
   ## 4. 伏笔与信息差
   ## 5. 暧昧与修罗场
   ## 6. 剧情精要
   ## 7. 衔接设计

   🚨 **格式禁忌**：
   - **严禁重复任何节标题**："## 1. 基础元信息"、"## 2. 张力与冲突"等在每章中只能出现一次
   - **严禁**在"基础元信息"中重复写"第X章 - 标题"
   ...
```

**改进**：
- ✅ 明确列出所有7个节
- ✅ 强调"每个节标题只能出现一次"
- ✅ 在格式禁忌中明确禁止重复任何节标题

### 修复2：强调每章必须包含所有节

**修复后的Prompt**：
```
2. **完整性铁律**：
   - 禁止任何省略（如"..."或"略"）。
   - 每章至少800字详细描述。
   - 必须包含【匠心思维应用】。
   - **每个节都必须有内容，不能省略**

4. **批次要求**：
   - ...
   - **每章的7个节都必须完整，不能偷工减料省略后面的节**
```

**改进**：
- ✅ 明确要求"每个节都必须有内容"
- ✅ 在批次要求中强调"不能偷工减料省略后面的节"

### 修复3：添加节级别验证

**位置**：`novel_generator/blueprint.py:262-324`

**新增验证1：节标题重复检测**
```python
# 统计每个章节内节标题出现次数
section_header_pattern = r'^##\s+\d+\.\s+[\u4e00-\u9fa5]+'
for chapter_num in expected_numbers:
    # 提取章节内容
    chapter_lines = lines[start_line:end_line]

    # 统计节标题
    section_counts = {}
    for line in chapter_lines:
        match = re.match(section_header_pattern, line)
        if match:
            section_title = match.group(0)
            section_counts[section_title] = section_counts.get(section_title, 0) + 1

    # 检查重复
    for section_title, count in section_counts.items():
        if count > 1:
            repeated_sections.append(f"第{chapter_num}章中'{section_title}'重复了{count}次")
```

**新增验证2：节完整性检测**
```python
# 检查每个章节是否包含所有必需的节
required_sections = ["基础元信息", "张力与冲突", "匠心思维应用", "伏笔与信息差",
                   "暧昧与修罗场", "剧情精要", "衔接设计"]

for chapter_num in expected_numbers:
    # 提取章节内容
    chapter_text = '\n'.join(chapter_lines)

    # 检查每个必需的节是否存在
    for required_section in required_sections:
        if required_section not in chapter_text:
            missing_sections.append(f"第{chapter_num}章缺失: {required_section}")
```
