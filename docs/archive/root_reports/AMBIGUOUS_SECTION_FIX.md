# 🛠️ "暧昧与修罗场"节被省略问题的修复

## 问题描述

用户尝试生成第1-5章蓝图时，经过5次尝试仍然失败。验证器报告某些章节缺失"暧昧与修罗场"节：

```
第1次生成：
- 第2章缺失: 暧昧与修罗场
- 第3章缺失: 衔接设计

第2次生成：
- 第2章缺失: 暧昧与修罗场
- 第4章缺失: 暧昧与修罗场
- 第5章缺失: 暧昧与修罗场

第5次生成：
- 第2章缺失: 暧昧与修罗场, 剧情精要
- 第5章缺失: 暧昧与修罗场, 剧情精要
```

## 🔍 根本原因分析

### LLM的误解

通过分析日志文件，发现LLM在某些章节（特别是那些不涉及女性角色互动的章节）中直接省略了"暧昧与修罗场"整个节。

**原因**：Prompt中对这个节的说明是：
```
   ## 5. 暧昧与修罗场
   *   [涉及的女性角色互动]
   *   [或填写：本章不涉及]
```

LLM理解为："如果不涉及，可以省略整个节"

### 实际生成示例

**第2章生成内容**（缺失暧昧与修罗场）：
```
4. 伏笔与信息差
...

6. 剧情精要    ← 直接跳过了第5节！
...
```

**第3章生成内容**（包含暧昧与修罗场）：
```
4. 伏笔与信息差
...

5. 暧昧与修罗场    ← 保留了第5节
涉及的女性角色互动：林小雨站在人群外围...

6. 剧情精要
...
```

### 问题模式

- 当章节涉及女性角色（如林小雨）时，LLM会保留"暧昧与修罗场"节
- 当章节不涉及女性角色（如纯战斗章节）时，LLM倾向于省略这个节
- 这违反了"所有7个节都必须存在"的要求

## 🛠️ 修复方案

### 修复1：明确节标题不能省略

**位置**：`blueprint.py:600-605`

**修改前**：
```python
   ## 5. 暧昧与修罗场
   *   [涉及的女性角色互动]
   *   [或填写：本章不涉及]
```

**修改后**：
```python
   ## 5. 暧昧与修罗场
   *   **涉及的女性角色互动**：[描述女性角色互动，如林小雨、苏清雪等]
   *   **🚨 重要**：即使本章不涉及任何女性角色，也**必须保留此节**，并填写"本章不涉及女性角色互动"
   *   **格式要求**：
       - 如果涉及：详细描述互动内容
       - 如果不涉及：必须写"本章不涉及女性角色互动"（不能省略整个节）
```

**改进**：
- ✅ 明确说明即使不涉及也必须保留节标题
- ✅ 给出了具体的不涉及时的填写示例
- ✅ 使用**🚨 重要**标记引起注意

## 🎯 Prompt工程最佳实践

### 1. 明确性原则

**❌ 不够明确的提示**：
```

*   [涉及的女性角色互动]
*   [或填写：本章不涉及]
```

**✅ 明确的提示**：
```

*   **涉及的女性角色互动**：[描述女性角色互动，如林小雨、苏清雪等]
*   **🚨 重要**：即使本章不涉及任何女性角色，也**必须保留此节**，并填写"本章不涉及女性角色互动"
*   **格式要求**：
    - 如果涉及：详细描述互动内容
    - 如果不涉及：必须写"本章不涉及女性角色互动"（不能省略整个节）
```

### 2. 重复强调原则

在Prompt中多次重复关键要求：
- 在节定义中说明
- 在格式禁忌中强调
- 在完整性铁律中重申
