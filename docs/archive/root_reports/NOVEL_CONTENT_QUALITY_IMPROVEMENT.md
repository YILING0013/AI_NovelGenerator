## 🔍 问题1：内容重复分析

### 1.1 当前去重机制

**位置**: `novel_generator/finalization.py:29-69`

```python
def check_chapter_similarity(new_chapter: str, previous_chapter: str, threshold: float = 0.15) -> tuple:
    # 按段落拆分，检测前80字符是否重复
    fingerprint = new_para[:80]
    if fingerprint in prev_para or prev_para[:80] in new_para:
        duplicate_segments.append(new_para[:60] + "...")
```

**局限性**:
- ❌ 只检测**相邻章节**之间的重复
- ❌ 只检查段落**前80字符**的匹配
- ❌ 无法检测**章节内部**的重复
- ❌ 无法检测**语义重复**（不同表述但相同含义）
- ❌ 阈值15%可能过高

### 1.2 重复内容的类型

#### 类型1：完全重复
```markdown
第2章：张昊在窑厂发现废料价值...
第3章开头：张昊在窑厂发现废料价值...
```

#### 类型2：语义重复
```markdown
第一次：张昊眼中，赵四的手臂像一件烧坏的瓷器
第二次：张昊用金瞳审视，将赵四的手臂看作劣质陶俑
```

#### 类型3：场景重复
```markdown
第2章：窑厂的高温与粉尘描写
第5章：炼器堂的火焰描写（使用相同形容词和句式）
```

### 1.3 改进方案

#### 改进1：增强去重算法

**新增功能**:
1. 章节内重复检测
2. 语义相似度检测（使用向量嵌入）
3. 跨章节重复检测（不只是相邻章节）
4. 模式重复检测（相同句式结构）

**代码实现建议**:

```python
from sentence_transformers import SentenceTransformer
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

class EnhancedDuplicateDetector:
    """增强的去重检测器"""

    def __init__(self):
        # 使用中文语义相似度模型
        self.model = SentenceTransformer('paraphrase-multilingual-MiniLM-L12-v2')

    def detect_semantic_duplicates(self, content: str, threshold: float = 0.85) -> list:
        """检测语义重复"""
        # 按段落拆分
        paragraphs = [p.strip() for p in content.split('\n\n')
                     if p.strip() and len(p.strip()) > 50]

        if len(paragraphs) < 2:
            return []

        # 生成嵌入向量
        embeddings = self.model.encode(paragraphs)

        # 计算相似度矩阵
        similarities = cosine_similarity(embeddings)

        # 找出重复对
        duplicates = []
        for i in range(len(paragraphs)):
            for j in range(i + 1, len(paragraphs)):
                if similarities[i][j] > threshold:
                    duplicates.append({
                        'para1_index': i,
                        'para2_index': j,
                        'similarity': similarities[i][j],
                        'content1': paragraphs[i][:100] + "...",
                        'content2': paragraphs[j][100] + "..."
                    })

        return duplicates

    def detect_pattern_duplicates(self, content: str) -> list:
        """检测句式模式重复"""
        import re

        # 提取句式模式（去掉具体名词、动词）
        lines = content.split('\n')
        patterns = {}

        for line in lines:
            if len(line.strip()) < 20:
                continue

            # 创建句式指纹
            pattern = self._create_sentence_pattern(line)

            if pattern not in patterns:
                patterns[pattern] = []
            patterns[pattern].append(line.strip())

        # 找出重复的句式
        duplicates = []
        for pattern, instances in patterns.items():
            if len(instances) > 2:  # 同一句式出现超过2次
                duplicates.append({
                    'pattern': pattern,
                    'count': len(instances),
                    'instances': instances[:3]  # 只保留前3个示例
                })

        return duplicates

    def _create_sentence_pattern(self, sentence: str) -> str:
        """创建句式模式指纹"""
        # 保留结构词，去掉内容词
        structure_words = ['的', '了', '在', '是', '有', '和', '与', '从', '到',
                          '被', '把', '让', '使', '像', '如同', '仿佛']

        pattern = []
        words = sentence.split()

        for word in words:
            if word in structure_words:
                pattern.append(word)
            elif re.match(r'[，。！？；：、]', word):
                pattern.append(word)
            else:
                pattern.append('[CONTENT]')

        return ' '.join(pattern)
```

### 2.2 主角的"修补天道"思维模式

4. **提前准备** - 时刻为更高的境界做准备

### 方案3：Prompt改进

**新增Prompt模块**:

```markdown
### 战斗逻辑生成指南（敌人境界三倍以上时）

**核心原则**：主角不靠蛮力，靠"准备"和"理性"

**必须包含的要素**：
1. **信息优势**（0.5秒内完成）
   - 瞬间解析敌人的功法结构
   - 找出"只有我能看到的缺陷"

   - "我推演过这种情况下127次"
   - "在藏书阁见过这个功法的残本"
   - "昨天刚观察到这个缺陷"

3. **风险对冲**（不是无敌）
   - 选择最优方案，但有代价
   - 主动承受可计算的伤害
   - 将伤害转化为资源

4. **修复思维**（独特视角）
   - 将攻击视为"修复契机"
   - 将敌人的力量视为"材料"

- ❌ 主角突然开挂，境界提升
- ❌ 敌人变蠢，送人头
- ❌ 战斗过程一帆风顺
- ❌ 主角零代价获胜
```
