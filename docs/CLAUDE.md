# docs 模块文档

## 模块职责

docs模块负责AI小说生成工具的完整文档体系，包括用户手册、开发指南、API参考和部署文档，为不同角色的用户提供全面的使用和开发支持。

## 文档结构

### 核心文档文件

#### README_NEW.md
- **用途**: 项目总体介绍和快速开始指南
- **目标用户**: 新用户和潜在贡献者
- **内容范围**: 项目概述、功能特性、安装指南

#### DEVELOPMENT.md
- **用途**: 开发环境和贡献指南
- **目标用户**: 开发者和贡献者
- **内容范围**: 开发环境搭建、代码规范、提交流程

#### USER_MANUAL.md
- **用途**: 详细用户操作手册
- **目标用户**: 最终用户
- **内容范围**: 功能使用、配置说明、常见问题

#### API_REFERENCE.md
- **用途**: API接口详细文档
- **目标用户**: 开发者和集成者
- **内容范围**: 接口规范、参数说明、示例代码

#### DEPLOYMENT.md
- **用途**: 部署和运维指南
- **目标用户**: 系统管理员和运维人员
- **内容范围**: 环境要求、部署步骤、维护指南

#### DOCUMENTATION_INDEX.md
- **用途**: 文档导航和索引
- **目标用户**: 所有用户
- **内容范围**: 文档结构、快速导航、更新日志
